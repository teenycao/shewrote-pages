中国古代才女

她写过

而记录可以证明

中国古代到底有多少女性诗人?过去并非无人考证,但答案从不在大众的视野里: 它散落在难读的学术数据库与不标性别的开源诗词库之间—— 从来没有一份贯通历代、单独辑出的女性诗人合集。

于是我们做了两件事:先从七十六万首古典诗词中把她们逐一考证出来, 建成开放数据集 SheWrote ↗; 再把数据集做成你眼前这个网站——把她们一个个找回来,放在能被看见的地方。

数字
911位女性诗人,站内收录
10,364首她们的诗,去重后

其中 903 位从七十六万余首古典诗词中交叉考证找回,另 8 位据正史、乐府、别集等公版文献补录(逐首标注出处、另计)。

这是下界,不是全部

我们据以考证的古典诗词共 763,541 首,其中 11,481 首可归属为女性所作——占 1.50%。这是下界,不是全部: 它只算了「能从诗里稳妥考证出是女性」的部分,仅以「某氏」留名的就有 376 首;更多人连一个字都没能留下。 我们不把所有的缺席都归因于抹去:有的是历史真的没有记下她, 有的只是现存数据尚未覆盖到她。但无论哪一种,找回来的每一个名字,都是证明。

补录的 8 位据公版文献人工录入、逐首标注出处;另计,不并入上面的 903 位 / 1.50% 语料口径。

我们是怎么做的

以哈佛中国历代人物传记数据库(CBDB)↗的性别与亲属记录, 交叉标注开源古典诗词,逐名考证、逐首归属;明清部分参考 明清妇女著作数据库 ↗。 方法论、复现步骤与全部数据已在 GitHub ↗ 开源,任何人都可以查看、验证与引用。

原则只有一条:标记,不猜测。无法考证的,宁可不收。

几个口径说明
  • 统计口径:11,481 为上游原始记录数(同一唐宋作品可能在两个上游诗歌库各存一份);跨库去重后 10,277 首,这是交叉考证的语料口径。站内连 87 首补录合计展示 10,364 首。
  • 站点显示使用简体;数据集本体保留繁体原貌——记录存真,展示从众。
  • 档案缺失的字段(籍贯、生卒等)直接不展示,不做推测填充。
  • 每人页首的代表句默认由程序自动摘取,可能与你心中她的名句不同;被 3 位以上读者在她诗里划出的句子,会顶替程序摘句并标注「读者划出」。
  • 「亲缘」来自 CBDB 亲属记录。才女常成对出现——母女、姊妹、姑侄,她们是彼此最早的读者。
勘误与反馈

这样一份跨越两千年的记录,难免有错——性别归属、亲缘方向、繁简转换、诗作系名…… 发现任何一处,都请告诉我们:每一条勘误,都让她们的记录更准确一点。

说了真的会改——每一个被找回的名字都公开在修订时间线。如果你想知道哪些名字暂不在站内、又为什么,可先看常见问题。你在站内搜索却没有结果的名字,我们也都记下了——它们是下一轮核实的线索。

也不只是勘误——对这个网站的改进想法,或者你觉得还有哪些「看不见的女性」的数据值得挖掘、值得被看见,都欢迎说给我们听。

在 GitHub 提交勘误 ↗, 或到公众号「对抗虚无」留言告诉我们。
